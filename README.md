
# Portfolio Website

## Structure

```
Portfolio-Website/
	├── assets/           # Images, icons, and static files
	├── sections/         # Modular HTML sections (about, projects, etc.)
	├── index.html        # Main entry point
	├── README.md         # Project documentation
```

## How it works

- The main page loads content dynamically from the `sections` folder for each navigation tab.
- All images and icons should be placed in the `assets` folder.
- To update a section, edit the corresponding HTML file in `sections/`.

## Customization

- Add new sections by creating new HTML files in `sections/` and updating the navigation in `index.html`.
- Update your icon by replacing `assets/icon.png`.
